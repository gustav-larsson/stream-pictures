help_outline GUIDE Guide to add stream pictures to OBS Be a tier 3 subscriber to Caiyth (other payment methods will be implemented)

Use the configurator and save any values (values doesn't work yet but will give users access to send you pictures)

Add a browser source in OBS.

Add the url: https://stream-pictures.web.app/picture-viewer?user={{user?.login}}

Make sure the custom css contains:
body {{ '{' }} background-color: rgba(0, 0, 0, 0); margin: 0px auto; overflow: hidden; {{ '}' }}

Send links to other Streamers with the Send Link Feature.

Decide what to show on stream with the Picture List Feature.
Accepting anything in the Picture List Feature show the picture instantly in the Picture Viewer.
